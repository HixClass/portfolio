    기본 유형 선택

    현재 모든 웹에서 가장 선호하는 게시판을 추천

    1. 메뉴 -> 태그검색
    2. 기본적으로 제공하고 있는 게시판 플랫폼 제공
    3. 빠른 게시판 제공

    태그 검색

    태그검색을 통해 원하는 이미지가 있는지 확인할 수 있습니다.

    1. 메뉴 -> 태그검색
    2. 검색란에서 키워드 검색 --- Ex) 반응형, 작성자, 작성날짜 etc.
    3. 기본 게시판, 공유게시판, 제작 게시판의 태그를 참조하여 이미지 리스트를 보여드립니다.

    나만의 게시판

    원하는 게시판 모양 요소를 선택하여 게시판을 만들 수 있습니다.

    1. 메뉴 -> 나만의 게시판
    2. 게시판 요소를 선택
    3. 선택 결과 중 다양한 플랫폼 중에서 가장 적합한 구조를 선택
    4. 게시판 설정 후 만들기

    공유 게시판

    사용자들의 아이디어를 배포하는 공간

    1. 메뉴 -> 공유게시판
    2. 공유하고자 하는 게시판을 등록 또는 검색
    3. 이미지와 상세 목록을 확인하여 적합합 구조가 있으면 게시판 클릭!
